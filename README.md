# 🌬️ CLIMATIZA THEREZINA

Landing page desenvolvida para a empresa **Climatiza Therezina**, com foco em apresentação dos serviços e contato rápido via WhatsApp.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 📱 Funcionalidades

* Layout responsivo (funciona em celular e computador)
* Botão direto para WhatsApp
* Link para Instagram
* Avaliação no Google
* Seção de produtos (climatizadores)
* Animações suaves ao rolar a página

---

## 📂 Estrutura do projeto

```
index.html
style.css
script.js
/imagem
```

---

## 🌐 Deploy

O site está hospedado no Vercel e atualizado automaticamente via GitHub.
